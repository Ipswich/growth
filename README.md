# growth
Raspberry Pi/Arduino based grow room controller. Mostly designed to run on an iPhone 8, but still functional on pretty much any other platform.

### Software:
* Node.js
* Johnny-Five
* Pug with Bootstrap
* MariaDB

### Computational Hardware: 
* Raspberry Pi 4, 2GB
* Arduino Mega

### Sensors:
* Bosch BME280

### Outputs:
* 8-Channel Relay
* 2 PC817X
